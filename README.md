# 2DPathFinderUnity
Easy 2D path finding for Unity

Features: 
1. Easy to use
2. Example included
3. Smart mode
4. Facing toward path (optional)
5. Avoid colliders
6. Find another way if there are collider on it's way (coming soon! if reach 100 watcher or $10 donations)
